# Comandos utilizados para criar este repositório

## Criando repositório no Github

1. Criei a pasta do repositório que eu queria em meu computador
2. Criei o repositório no github
3. Criei o readme inicial do projeto e o .gitignore
4. Lista dos comandos a serem executados
    - `git init`
    - `git add .`
    - `git commit -a -m "commit inicial"`
    - `git remote add origin git@github.com:DougAugSilva/modelos_ml_supervi.git`
    - `git branch -M main`
    - `git push -u origin main`

## Criando ambiente virtual python

1. Ajustando a versão do python: 
    - `asdf python version`
    - `asdf local python 3.14.2`
2. Definindo ambiente virtual venv
    - `python -m venv .venv`
    - `source .venv/bin/activate`
